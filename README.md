# Student Manager (Python)

A command-line Student Manager application built with Python. This project demonstrates fundamental Python programming concepts including functions, dictionaries, file handling, and JSON persistence.

## Features

- Add new students
- View all students
- Search for a student
- Update student details
- Delete student records
- Automatically save data to a JSON file
- Automatically load saved data when the program starts

## Technologies Used

- Python 3
- JSON
- File Handling

## Concepts Practiced

- Functions
- Dictionaries & Nested Dictionaries
- Loops
- Conditional Statements
- JSON Serialization & Deserialization
- CRUD (Create, Read, Update, Delete)
- Exception Handling

## Future Improvements

- Input validation
- Student IDs
- Attendance tracking
- Grade management
- Graphical User Interface (GUI)
- Database integration (SQLite/MySQL)
